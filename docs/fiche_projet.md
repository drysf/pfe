## 3. Description du projet

Développement d'une machine virtuelle autonome hébergeant un **escape game
cybersécurité** composé de 5 défis progressifs :

1. Authentification par mot de passe faible
2. Injection SQL
3. Cryptographie (AES/César)
4. Stéganographie
5. Analyse réseau

Chaque défi débloque un fragment du flag final : `CTF{GROUPE3_2026}`

## 4. Technologies utilisées

- **Langage** : Python 3
- **Framework web** : Flask
- **Virtualisation** : Vagrant + VirtualBox / Docker
- **OS cible** : Ubuntu Server 22.04 LTS
- **Outils** : OpenSSL, Steghide, Wireshark, Netcat
